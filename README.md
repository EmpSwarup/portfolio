# My Portfolio Site made using React JS

## Tech/Features:

- React JS using Vite
- Vanilla CSS
- GSAP for preloader
- AOS for animations
- Unicons and Boxicons for icons
- Emailjs for mails
- React Swiper for swiping
- Vercel analytics and speed insights for analytics
- Deployed using Vercel
